The first colored 'text' element has the font-size set, so the animation of the parent 'g' only affects the fill color.

The second has the fill set and font-size set, so no inherited values are used. The font-size and fill color stay constant.

The third colored 'text' element has neither of these properties specified and thus inherits both animated values - the fill color changes and the text grows in size.
