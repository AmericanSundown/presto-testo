Test passes if all the different cursor behaviour is correct according to the description.

  1. Move the cursor to the top left corner, in the white area. This is the default cursor.
  2. Move the cursor over the text at the top of the example. The cursor changes to the text cursor.
  3. Move the cursor to the blue link text - the cursor changes to the pointer cursor.

Now, move the cursor in turn to each of the gray rectangles. From top to bottom in the first row, the cursor should change to:

  1. A crosshair or other 'accurate positioning' cursor
  2. The 'default' cursor, as noted above
  3. The 'pointer' cursor, as noted above
  4. A cursor indicating movement, such as panning

Now from top to bottom in the second row, the cursor should change to:

  1. The 'text' cursor, as noted above
  2. A 'wait' cursor
  3. A 'help' cursor
  4. A special cursor which looks like a small magnifying glass. This is a downloaded image cursor.

Moving to the bottom-rightmost of the eight red triangles, and moving around them clockwise, the cursor should change to:

  1. SouthEast-resize
  2. South-resize
  3. SouthWest resize
  4. West-resize
  5. NorthWest-resize
  6. North-resize
  7. NorthEast-resize
  8. East-resize

Lastly, move the cursor to the target in the bottom-right of the test. The cursor must not change to the 'pointer' cursor, but instread to the special magnifying glass cursor.
