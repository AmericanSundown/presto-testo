DOM 2 - CSS2 Interface

These tests are written to verify that Opera supports the CSS2 Interface of DOM 2.
The CSS2 Interface is accessed throught the ElementCSSInlineStyle Interface of DOM 2, that is through the style attribute of HTML-elements like this:
document.getElementById("someid").style.color = "red";
