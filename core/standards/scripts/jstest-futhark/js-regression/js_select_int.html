Select Object Test - Interactive

The popup window is for reportrs from the different event handlers. Use the select boxes to test the eventhandlers onblur, onchange, onfocus.

  1. onblur(): When a form element loses focus or when a window or frame loses focus.
  2. onchange(): When the field loses focus and its value has been modified.
  3. onfocus(): When someone clicks or uses the TAB to get focus in the form element.
