History Object Test - Interactive

The popuup window with the url field is your browsing window. Enter a few urls in the address field, so the window object gets a few entries in its history list. Then use the links below to test the different methods of navigating in this history.

The other popup window is for the different navigation methods to report what is the new values of the current, previous and next properties after one of the methods below has been issued.

User interaction

On popup window

  1. Go back in history using history.back() Loads the previous URL in the history list.
  2. Go forward in history using history.forward() Loads the next URL in the history list.
  3. Go back in history using history.go( -1 ) Loads the previous URL from the history list.
  4. Go forward in history using history.go( 1 ) Loads the next URL from the history list.

On this window

You should have been to one other page before using this page in order to get these tests working. Also, after loading this page, enter a new url, load this page and then go back again to this page. Now, you should have both entries in future and back history, so to speak, and should be able to use the methods below.

  2. Go forward in history using history.forward() Loads the next URL in the history list.
