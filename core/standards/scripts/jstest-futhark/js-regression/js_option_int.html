Option Object Test - Interactive

The second option in the select should be selected by default, and display the text "Default selected" in the drop down list.

User interaction

  1. Add an option to the select box. Report of the event should occur in the popup window. There should be a new option available in the drop down list called 'Another option text'.
  2. Add yet another option to the select box. Report of the event should occur in the popup window. There should be a new option available in the drop down list called 'Yet another option text', and it should be selected by default in the drop down list.
  3. Remove the last option. Reports on the event is given in the popup.
  4. Add even yet another option to the select box and use selectedIndex on the select to select the now second last option. Adds "Even yet another" opton to the select, and uses the selectIndex property of the options array to set the selected options to the second last item. Thus, the last option before adding another.
  5. Change the text of the first item. The text displayed in the list box should change. The selected item should not change.
