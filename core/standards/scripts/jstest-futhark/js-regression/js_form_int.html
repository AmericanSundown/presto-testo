Form Object Test - Interactive

A popup window will show when the first event is triggered, e.g. when you reset the form. The popup window is for reports from the different event handlers.

User interaction

  1. reset() Resets all form values ( texts, checkboxes etc ) to their default values. The same behavior should be expected when clicking the reset button. The onreset() event handler should be triggered.
  2. submit() Submits the form. The same behavior should be expected when clicking the submit button. The onsubmit() event handler should be triggered.
