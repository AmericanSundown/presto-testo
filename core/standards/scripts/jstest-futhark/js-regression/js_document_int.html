User interaction

Event Handlers

  1. onclick, when an object on a form is clicked. Click the button.
  2. ondblclick, when the user double-clicks a form element or a link. Try double clicking on the button.
  3. onkeydown, onkeypress, onkeyup. Press a key, e.g. 'd' in Opera to go to the next block of text. The event handlers should trigger in the order listed here.
  4. onmousedown, when you press your mousebutton.
  5. onmouseup, when you let go of your mousebutton.
