Button Object Test - Interactive

A popup window will show when the first event is triggered, e.g. when you click the button. The popup window is for reports from the different event handlers.

User interaction

  1. focus(), the button receives the focus. The onfocus() event handler should trigger.
  2. click(), the button is clicked. The onclick() event handler should trigger.
  3. blur(), the focus is removed from the button The onblur() event handler should trigger. ( it needs to have focus first )
  4. Click the button. The are three event handlers that should now trigger, reporting in the popup window: onmousedown(), onmouseup() and onclick(), onmousedown() should trigger before onmouseup(). If the button didn't have focus already, onfocus() should also be triggered.
