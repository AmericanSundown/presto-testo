First test

Type the following into a blank window in your web browser: 
    javascript:'hello world'
and press ENTER. The window should be updated and should contain the string "hello world" (without any quotation marks).

Second test

Create a new bookmark containing the following code: 
  javascript:alert('Cookie for this site:'+document.cookie)
Go to http://www.google.com and then select the new bookmark; it should pop up an alert showing you the cookie that Google set.
