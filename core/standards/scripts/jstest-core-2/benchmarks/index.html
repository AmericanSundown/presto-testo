JavaScript benchmark programs

This is a collection of simple benchmark programs. When they are run inside Opera, they test (a) the performance of the ECMAScript engine and (b) the performance of the JavaScript scheduler. From these benchmarks it is generally impossible for you to tell where the time goes -- you need access to the Opera source code for that. (For example, in Opera 5 and Opera 6 a great deal of time was lost in the scheduler.) But these will give you at least some idea where Opera is, relative to the competition.

Notes

  • For accurate measurement, try to reduce noice by disabling all other programs on your system
  • Some care has been taken to reduce environmental effects (like startup overhead and timer granularity)
  • Tests may take up to a few minutes in some cases but most complete in less than 30 seconds
  • Bear in mind when you view results that these are micro-benchmarks, usually testing only a few features
  • The iBench tests have been extracted from a possibly older version of the Ziff-Davis iBench test suite
