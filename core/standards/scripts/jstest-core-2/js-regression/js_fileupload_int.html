FileUpload Object Test - Interactive

A popup window will show when the first event is triggered, e.g. when you click in the textfield. The popup window is for reports from the different event handlers.

User interaction

  1. focus(), the checkbox receives focus The onfocus() event handler should trigger.
  2. blur(), focus is removed from the checkbox. The onblur() event handler should trigger. ( it needs to have focus first )
  3. select() Selects the input area of the file upload field.
  4. onchange(). Event handler should be triggered when the field loses its focus and its value has been modified. Try modifiying the string in the text field, and then remove focus from the text field and button ( the fileupload element ).
