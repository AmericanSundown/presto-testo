The popup window is for reportrs from the different event handlers for the two link objects.

www.opera.com

Imagemap, link object is an area element

User interaction

  1. Anchor & Area: onclick(). Triggered when clicking the link.
  2. Anchor & Area: onmouseover(). Triggered when hovering the mouse over the link (area).
  3. Anchor & Area: onmouseout(). Triggered when the mouse leaves the link area.
  4. Anchor: ondblclick(). Triggered when double clicking on the link
  5. Anchor: onkeydown(), onkeypress(), onkeyup(). Triggered when a key is pressed on the link. e.g. navigating between the two links with 'a' and 'q' in Opera. Should be triggered in the order listed here.
