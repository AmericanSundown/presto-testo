Submit Object Test - Interactive

The popup window is for reports from the different event handlers. Click the links below and test the key event handlers by entering something in the submitfield. onblur, onclick, onfocus.

  1. onblur(): When a form element loses focus or when a window or frame loses focus.
  2. onclick(): When the form object is clicked.
  3. onfocus(): When someone clicks or uses the TAB to get focus in the form element.

User interaction

  1. Click the submit button
  2. Remove focus from the button ( if it has focus )
  3. Get focus on the submit button
