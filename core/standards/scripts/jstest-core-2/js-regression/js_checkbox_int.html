Checkbox Object Test - Interactive

A popup window will show when the first event is triggered, e.g. when you click the checkbox. The popup window is for reports from the different event handlers.

User interaction

  1. focus(), the checkbox receives focus The onfocus() event handler should trigger.
  2. click(), the checkbox is clicked The onclick() event handler should trigger.
  3. blur(), focus is removed from the checkbox. The onblur() event handler should trigger. ( it needs to have focus first ).
  4. Click the checkbox. onclick() should trigger. If it didn't have focus, onfocus() is also triggered.
