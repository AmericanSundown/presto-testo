Radio Object Test - Interactive

The popup window is for reports from the different event handlers. Click the links below to test the key event handlers onblur, onclick and onfocus.

When clicking the different buttons in the radio button group, only one should be selected at a time. The previous button that had focus, should trigger an onblur(), while the one gaining the click should report an onfocus() followed by a onclick().

User interaction

  1. Click the first radio button The first radio button on the page should be selected.
  2. Remove focus from the first radio button If the first radio button had focus, it should be removed.
  3. Get focus on the first radio button The first radio button should gain focus, but not change value ( selected/not selected ).
