The <embed> element:

This is used to load videoclips etc. Therefore not applicable for Rubik without a plugin.
The testcase is still included and can be viewed using Windows.
The correct plugin for .avi files must be installed and configured.

The code for this is: <embed src="dynsrc.avi" align="center" autostart="true" height="100" width="100" loop="true" controls="true"></embed>
