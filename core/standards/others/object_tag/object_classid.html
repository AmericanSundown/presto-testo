The Flash animation is displayed using the CLASSID, CODEBASE and CODETYPE attribute of the OBJECT element and the NAME and VALUE attributes of the PARAM element. The Flash animation can not play without the PARAM elements in this case.

This is the code:

<OBJECT classid="clsid:D27CDB6E-AE6D-11cf-96B8-444553540000"
codebase="http://active.macromedia.com/flash2/cabs/swflash.cab#version=4,0,0,0"
codetype="application/x-shockwave-flash"
WIDTH=480 HEIGHT=60>
<PARAM NAME=movie VALUE="banner2001.swf">
</OBJECT>
