Extended cache management

Edit opera.ini: Add people.opera.com to "trusted domains".

[OEM]
Never Flush Trusted Servers=people.opera.com

 

Load the start page. This is what you do with each link on the test page:

  1. Click (link on start page) and remember/write the text in the image.
  2. Empty the cache (in the browser, do not delete the files)
  3. Click (link on start page) Do not reload, or use the history.
  4. Adjust the system clock two days forward.
  5. Click (link on start page) not history and not reload.
  6. Adjust the system clock back.

Expected results

The images have "Expire:" set 24 hours into the future. Adjusting the system clock two days forward will make all the images "stale".

Without never-flush

Never changing image

When revisiting the link after the cache has been flushed, the image is the same as last time.

When revisiting the link after adjusting the system clock, the image doesn't change. But it should be loaded from server.

Always changing image

When revisiting the link after the cache has been flushed, the image changes.

When revisiting the link after adjusting the system clock, the image changes.

With never-flush

Never changing image

When revisiting the link after the cache has been flushed, the image is the same as last time.

When revisiting the link after adjusting the system clock, the image doesn't change. But it should be loaded from server.

Always changing image

When revisiting the link after the cache has been flushed, the image is the same as last time.

When revisiting the link after adjusting the system clock, the image changes.
