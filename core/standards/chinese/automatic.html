Automatic encoding detection

1) Start Opera

2) Go to a page using ISO-2022-JP encoding: http://t/imported/peter/unicode/html/excess2022.html

3) Page should display with encoding ISO-2022-JP

4) Go to page with an unsupported encoding: http://t/imported/peter/unicode/autodetect/zh/iso2022cn.html

5) RefUI should return infoprint "Encoding not supported"

6) Go to a page without encoding specified: http://t/imported/peter/unicode/

7) RefUI should return infoprint "Encoding not detected"
