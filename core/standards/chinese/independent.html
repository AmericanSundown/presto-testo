Independent encoding setting per window

1) Start Opera

2) In RefUI, go to View > Charset > Override

3) Go to File > New window

4) Go to View > Charset > General (Chinese)

5) Go to Window and choose the window you performed step 2 on

6) Go to View > Charset and verify that the charset is still set to Override

7) Go to Window and choose the window you opened in step 3

8) Go to View > Charset and verify that the charset is still set to General (Chinese)
