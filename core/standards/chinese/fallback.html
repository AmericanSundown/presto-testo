Fallback encoding of browser

In RefUI, the encoding used for testing fallback encoding is hardcoded.

1) Start Opera

2) In RefUI, go to Test > Test: GetFallbackEncoding

3) RefUI should return the infoprint: "windows-1252"

3) In RefUI, go to Test > Test: SetFallbackEncoding

4) In RefUI, go to Test > Test: GetFallbackEncoding

5) RefUI should return an infoprint with new fallback encoding: "JIS_Encoding"
