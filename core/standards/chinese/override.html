Set window to override encoding

In RefUI, the encoding used when testing "override encoding" has been hardcoded.

1) Start Opera

2) Go to a page with a supported encoding (UTF-8): http://t/imported/peter/unicode/html/excess2022.html

3) RefUI should return infoprint: "Encoding detected: UTF-8"

4) In RefUI, go to View > Charset > Override

5) Page should now display lots of squares due to wrong encoding being used
