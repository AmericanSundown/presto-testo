This section has a testsuite for the <canvas> element feature from HTML5. The
specification is available online:

  http://www.whatwg.org/specs/web-apps/current-work/#canvas

Most of the tests can be found in the subdirectories of "2d" which is in
"getContext".

Other <canvas> testsuites:
  http://canvex.lazyilluminati.com/tests/tests/
  http://hixie.ch/tests/adhoc/html/canvas/
