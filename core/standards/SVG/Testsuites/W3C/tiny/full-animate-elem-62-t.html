The test validates the various possibilities for the end attribute value: no specified value, offset value, event base value, sync base value, indefinite value, repeat value, accessKey value and wallclock.

Quick description to run test in gogi:

all red rectangles should turn green in order from left to right, at the given second.

The sequences to the left starts at load. The sequences to the right are started like this:

  • "event base": click on corresponding red rectangle.
  • "accessKeys()": choose "Enable access key mode" in menu, click on svg image to focus it, and press "a".
